A comprehensive real-time analytics dashboard for tracking metrics across multiple platforms.

## Dashboard Metrics

YouTube: Subscriber count, total views, video count
Mailchimp: Email list size and contacts
Memberful: Active subscriptions, MRR, revenue metrics
Discord: Online members, total members, server insights (500+ members)
Google Analytics: Website traffic and user behavior


## Built With
Backend: Node.js + Express
Frontend: Vanilla JavaScript + Chart.js
Storage: JSON file-based history
Deployment: Replit-ready, portable to any Node.js host


## Features 
Secure API key management via environment variables
Automatic data refresh (30-min intervals)
Manual snapshot saving for historical tracking
Zero-config deployment on Replit
